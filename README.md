# 🌿 올본한의원 처방 관리 시스템

한의원 처방, 재고, 입출고 내역을 관리하는 웹 기반 시스템입니다.

## 🎯 주요 기능

- **처방 관리**: 쌍화탕, 부기탕 등 빠른 처방 및 상세 처방 작성
- **재고 관리**: 실시간 약재 재고 현황 및 알림
- **입출고 내역**: 자동 재고 업데이트 및 이력 관리
- **대시보드**: 오늘/이달 통계 및 주요 지표

## 🚀 기술 스택

- **Frontend**: HTML, CSS, JavaScript (Vanilla)
- **Backend**: Google Apps Script
- **Database**: Google Sheets

## 📦 설치 및 설정

자세한 설정 방법은 `SETUP_GUIDE.md` 파일을 참조하세요.

### 간단 요약:

1. Google Sheets 생성 (3개 시트: 처방내역, 재고현황, 입출고내역)
2. Apps Script 코드 배포 (`google-apps-script/Code.gs`)
3. Web App URL을 브라우저 콘솔에서 설정
4. 완료!
